# Cursor Backend I — Coderhouse

Este repositorio reúne **ejercicios y proyectos** desarrollados durante algunas clases del curso **Cursor Backend I** de [Coderhouse](https://www.coderhouse.com/). El profesor a cargo del módulo de Cursor fue **Maximiliano Martin**.

Como **referencia del material visto en clase**, el docente fue publicando el código en el repositorio [Nirbeat/coderBack1_81200](https://github.com/Nirbeat/coderBack1_81200) (comisión 81200), con ramas por encuentro; por ejemplo la rama [`clase1`](https://github.com/Nirbeat/coderBack1_81200/tree/clase1).
